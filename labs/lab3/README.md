### How to Setup

```sh
virtualenv -p python3 my-venv
. my-venv/bin/activate
pip3 install -r requirements.txt
```

### How to run Chat server

```sh
python3 chat_server.py
```

### How to run a Chat client

```sh
python3 chat_client.py [username]
```
